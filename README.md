# Teacher Website

موقع بسيط للمدرس بواجهة عربية ومتصل بقاعدة بيانات SQLite باستخدام Python standard library فقط.

## المميزات
- عرض بيانات المدرس من قاعدة البيانات.
- عرض الكورسات والإعلانات بشكل ديناميكي.
- نموذج تواصل يحفظ استفسارات الطلاب في قاعدة البيانات.
- API بسيطة لتسهيل التوسع لاحقًا.
- لا يحتاج إلى تثبيت مكتبات خارجية.

## التشغيل
```bash
python3 app.py
```

ثم افتح المتصفح على:

```text
http://127.0.0.1:5000
```

## قاعدة البيانات
سيتم إنشاء ملف `teacher_site.db` تلقائيًا عند أول تشغيل.

## نقاط تطوير مستقبلية
- إضافة لوحة تحكم للمدرس.
- تسجيل دخول للطلاب.
- رفع ملفات وواجبات.
- ربط الموقع بدفع إلكتروني أو حجز أونلاين.
